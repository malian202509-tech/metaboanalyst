# 胎盘氧化脂质组学数据分析标准作业流程（SOP·增强版）

**核心科学问题**：孕前正常 BMI 与孕前超重/肥胖 (Overweight/Obese) 孕妇，其分娩时胎盘的氧化脂质谱发生了何种特征性改变？

**研究定位**：探索性横断面观察性研究（Exploratory cross-sectional observational study），目标是"在调整核心临床混杂后，发现孕前体质量影响下胎盘氧化脂质网络的趋势性改变"，而非寻找诊断 biomarker。

---

## 阶段 0：元数据准备与样本均衡性诊断

**目标**：在开任何代谢组学分析之前，先把"两组人群本身是否可比"这件事讲清楚。这一步直接决定了第三阶段协变量清单与第四阶段敏感性分析怎么设计。

### 0.1 元数据表（Metadata table）整理

把所有样本的协变量整理成一张宽表，每行一个样本，每列一个变量。**最低限度需收集的字段**：

| 变量类别 | 字段 | 数据类型 | 备注 |
|---|---|---|---|
| 分组 | 孕前 BMI 分组（正常 / 超重肥胖） | 二分类 | 主分析的暴露因素 |
| 分组 | 孕前 BMI（连续值，kg/m²） | 连续 | 留作连续暴露的敏感性分析 |
| 临床 | 孕妇年龄 | 连续 | 强混杂 |
| 临床 | 产次 (parity) | 有序 | 初产 vs 经产，免疫表型差异大 |
| 临床 | 孕周 (gestational age, 周+天) | 连续 | 必须 |
| 临床 | 孕期增重 GWG (kg) | 连续 | 必须 |
| 分娩 | 分娩方式 | 三分类：顺产 / 择期剖宫产 / 急诊剖宫产 | **不要二分**，三种状态对应三种 COX-2 激活水平 |
| 分娩 | 是否临产 / 产程长度（h） | 连续 | 临产本身急性诱导前列腺素通路 |
| 胎儿 | 胎儿性别 | 二分类 | 胎盘表型存在性别二态性 |
| 胎儿 | 出生体重、是否巨大儿/FGR | 连续 + 二分类 | |
| 并发症 | GDM、子痫前期、胎窘 | 各二分类 | 协变量或剔除依据 |
| 取样 | 取样部位（中央 / 边缘；母体面 / 胎儿面） | 分类 | **SOP 必须确认全队列统一**；若不统一就当协变量 |
| 取样 | 离体到冻存时间（min） | 连续 | 氧化脂质 LOX/COX 离体后仍在反应，>30 min 谱图会漂 |
| 取样 | −80 ℃ 储存时长（月）或采样年份 | 连续 / 分类 | 与"批次"高度共线，二选一进模型 |
| 技术 | 上机批次号、进样顺序 | 分类 / 整数 | QC-based 漂移校正用 |

### 0.2 Table 1：基线均衡性描述

**做什么**：分组 × 协变量的描述性统计，目的是回答"两组人除了 BMI 不同之外，还有什么系统性不同？"。

**怎么报告**：
- 连续变量：若近似正态，报 `Mean ± SD`；偏态报 `Median (IQR)`；组间比较用 t 检验 / Wilcoxon。
- 分类变量：报 `n (%)`；组间比较用 χ² 或 Fisher 精确检验（任一格期望 <5 用 Fisher）。

### 0.3 标准化均数差（Standardized Mean Difference, SMD）

**为什么不只看 P 值**：小样本队列里 P 值会因为 n 小而"看起来没差异"（假阴性的不均衡），而实际两组的临床特征已经偏到不能忽略。SMD 不受样本量影响，是观察性研究里报告均衡性的金标准（见 STROBE / TRIPOD 指南）。

**怎么算**：

- 连续变量：

$$\text{SMD} = \frac{\bar{x}_1 - \bar{x}_2}{\sqrt{(s_1^2 + s_2^2)/2}}$$

- 二分类变量（比例 p）：

$$\text{SMD} = \frac{p_1 - p_2}{\sqrt{[p_1(1-p_1) + p_2(1-p_2)]/2}}$$

- 多分类变量：用 Mahalanobis 距离形式的多变量 SMD，或对每个水平做哑变量后逐个算。

**怎么解读**（行业通用经验阈值，源自 Austin 的倾向评分文献）：

| |SMD| | 含义 | 处理建议 |
|---|---|---|
| < 0.10 | 可忽略的不均衡 | 不必纳入模型 |
| 0.10 – 0.25 | 轻度不均衡 | 推荐纳入 GLM 作为协变量 |
| > 0.25 | 明显不均衡 | **必须**纳入协变量，且要做敏感性分析（如分层或匹配） |
| > 0.50 | 严重不均衡 | 警惕"无法用统计学完全修正"，在讨论部分必须明示 |

**输出**：一张 Table 1，列出每个协变量的 `正常组 vs 超重肥胖组` 描述统计 + P 值 + SMD 三列。**这张表里 |SMD| > 0.10 的所有变量，自动进入第三阶段的协变量清单**。

### 0.4 样本量与功效预估

- 报告每组的 n。
- 若 n < 20/组，明确在讨论里写"统计功效有限，结果以效应量与方向性为主而非显著性"。
- 若做了功效分析（如基于先验文献的 Cohen's d），写明假设与可检出最小效应。

## 阶段 1：原始数据质量控制与预处理

**目标**：在进入任何统计模型之前，把数据矩阵处理到"残差近似正态、特征尺度可比、低质量特征已剔除"的状态。**这一步是整套分析的地基**，被原 SOP 跳过。

### 1.1 QC 样本质控

**做什么**：把所有样本上机时按一定顺序穿插的 **池化 QC 样本（pooled QC）** 单独拿出来评估。

**评估指标**：
1. **QC 样本在 PCA 中的聚集度**：QC 应在 PC1–PC2 空间紧密聚成一簇，且位于所有真实样本的"质心"附近。如果 QC 自身散开，说明仪器漂移严重，必须做漂移校正。
2. **每个特征在 QC 中的 CV%**：
   $$\text{CV}\% = \frac{\text{SD}_{QC}}{\text{Mean}_{QC}} \times 100\%$$
   - **CV% ≤ 30%** 的特征：保留（MSI 推荐阈值）。
   - **CV% > 30%** 的特征：剔除，认为重复性不达标，不参与后续统计。
3. **TIC（总离子流）随进样顺序的稳定性**：画 TIC vs 进样序号图，若出现单调漂移或台阶式跳变，需配合 QC-RLSC/SERRF 校正。

### 1.2 特征级缺失与零值过滤

**80% 规则（modified 80% rule）**：

- 在**任一**分组内，某特征非零检出率 ≥ 80%，则保留。
- 若两组中都 < 80%，剔除（认为该特征在该队列基本不可检出）。
- 这一步先于插补，避免给一堆"几乎全缺失"的特征强行编故事。

### 1.3 缺失值插补

氧化脂质缺失值多源自"低于检出限（LOD）"，因此 **优先选 LOD 类插补，而非平均值**。

| 方法 | 适用情形 | 推荐度 |
|---|---|---|
| 半最小值（1/2 min per feature） | 缺失被认为是 LOD 截断 | ★★★★★（首选） |
| KNN 插补（k=5–10） | 缺失被认为是随机的 | ★★★★ |
| Random Forest（missForest） | 缺失机制混合 | ★★★★ |
| 均值/中位数插补 | / | ★（不推荐，会压缩组间差异） |

**报告要求**：在 Methods 写明插补方法、插补比例（按特征均值或全矩阵均值）。

### 1.4 归一化（Normalization）

消除"每个样本上机总量不一致"造成的整体偏移。

- **首选 PQN（Probabilistic Quotient Normalization）**：对生物体液和组织代谢组学最稳健。
- 备选：中位数归一化、总和归一化（TIC normalization）。
- **组织样本可同时做生物学归一化**：除以样本湿重 (mg) 或总蛋白浓度 (BCA 法)。氧化脂质强烈推荐归一化到蛋白浓度。

### 1.5 对数转换（Log transformation）

代谢组学浓度跨数个数量级，残差极偏态，必须 log 之后才能进 GLM。
- 推荐 **log2 或 log10**；含零或近零值用 **glog（generalized log）** 避免 −∞。
- 转换后画 QQ 图复核残差正态性。

### 1.6 缩放（Scaling）

| 方法 | 公式 | 适用 |
|---|---|---|
| Autoscaling (UV) | `(x − mean)/SD` | 所有特征同等权重，适合多元模型 |
| Pareto scaling | `(x − mean)/√SD` | **推荐**，兼顾大丰度与小丰度特征 |
| Range scaling | `(x − mean)/(max − min)` | 对离群敏感，少用 |

**典型组合**：`半最小值插补 → PQN → log2 → Pareto scaling`。

### 1.7 阶段产物

- "原始矩阵"（仅过 1.1–1.3）：用于浓度报告与可视化。
- "分析矩阵"（过完 1.1–1.6）：用于所有后续 PCA、批次校正、GLM、富集。
**两份矩阵分别归档，文章 Methods 必须写明每一步的顺序与参数**。

## 阶段 2：批次效应诊断与校正

**目标**：排除仪器漂移与跨年储存降解造成的系统性偏倚。

### 2.1 全局结构诊断

- **PCA 着色策略（三张图必出）**：
  1. 按"采样年份 / 批次"着色 → 看技术结构
  2. 按"进样顺序"着色（用连续色阶） → 看仪器漂移
  3. 按 QC vs 真实样本着色 → 看 QC 聚集度

- **判读**：
  - PC1 或 PC2 与批次明显共线 → 存在批次效应
  - QC 在主样本外形成单独簇 → 漂移严重
  - 沿进样顺序出现颜色梯度 → 进样漂移

### 2.2 批次—生物因素共线性检查（**关键，常被忽略**）

**做什么**：在校正之前，做一张 "批次 × BMI 分组" 的列联表，并算 Cramér's V。

**为什么**：如果 2019 年送检的样本几乎全是正常 BMI，2021 年几乎全是肥胖，那"年份"和"分组"完全共线，ComBat 会**连同生物学信号一起抹掉**。

**判读**：
- Cramér's V < 0.2：可放心校正。
- 0.2 – 0.5：警惕，建议把"批次"作为协变量进 GLM，而**不是**用 ComBat 抹掉。
- > 0.5：严重共线，禁用 ComBat；在讨论里坦诚批次与分组无法解耦，必要时分批次重做并比较一致性。

### 2.3 校正方法选择

| 场景 | 推荐方法 | 原因 |
|---|---|---|
| 按"年份/批次"的离散偏移，且批次与分组基本均衡 | **ComBat（经验贝叶斯）** / pycombat / inmoose | 经典，适合离散批次 |
| 同一批次内沿进样顺序的连续漂移 | **QC-RLSC**（QC 罗布斯样条校正）或 **SERRF**（随机森林） | 利用 QC 拟合漂移曲线，比 ComBat 更适合 LC-MS |
| 未知混杂、想模型化保留生物学方差 | **RUV-random / RUVg** | 适合复杂混杂 |

**实务建议**：
- **优先用 QC-based 方法处理仪器漂移**；
- 处理完漂移后再用 ComBat 抹年份；
- 校正参数（parametric vs nonparametric ComBat、是否带协变量 mod = BMI 分组）需在 Methods 详写。**带 mod 参数**能告诉算法"保护这个生物学变量不要被抹掉"。

### 2.4 校正后验证（三件套）

1. **PCA 重画**：按年份着色应混匀；按 QC 着色应聚集。
2. **QC 样本 CV% 应整体下降**，且整体方差不应被过度压缩（否则是"洗过头"）。
3. **方差解释结构**：比较校正前后 PC1/PC2 解释比例与组间差异方向，确保生物学信号没被一同抹掉。

## 阶段 3：队列构建与协变量策略

**目标**：定义清晰的分析队列与多重协变量调整方案。

### 3.1 分组定义

- **Group A（对照）**：孕前 BMI 18.5–23.9 kg/m²（按中国成人标准；若用 WHO 用 18.5–24.9）。
- **Group B（暴露）**：孕前 BMI ≥ 24（中国）或 ≥ 25（WHO）。
- **明确排除**：低体重（BMI < 18.5）、孕前 BMI 缺失、双胎、辅助生殖（如可避免）、产前已诊断的胎盘疾病（如前置胎盘出血）。

### 3.2 双轨协变量策略（**取代原 SOP "二选一"**）

不要二选一，**两条都跑、互为佐证**。

#### 主分析（首选）：策略二·协变量调整
保留所有合并症样本，在 GLM 中纳入以下协变量（强制项 + 阶段 0 SMD > 0.10 项）：

**强制纳入的协变量清单（建议 6+）**：
1. 孕妇年龄
2. 产次（初产 vs 经产）
3. 孕周（连续）
4. 孕期增重 GWG（连续）
5. 分娩方式（三分类）+ 是否临产
6. 胎儿性别
7. 离体到冻存时间
8. 主要并发症（GDM、子痫前期，各自二分类）
9. 阶段 0 中 |SMD| > 0.10 的其他变量

**模型形式**（概念性写法）：
```
log2(metabolite) ~ BMI_group + age + parity + GA + GWG
                  + delivery_mode + labor_onset
                  + fetal_sex + ischemia_time
                  + GDM + PE + [其他 SMD>0.10 项]
```
提取 `BMI_group` 系数对应的 P 值与效应量（log2 fold change 及 95% CI）。

#### 敏感性分析：策略一·纯净亚组
**剔除所有合并症样本**（GDM、PE、巨大儿、FGR、胎窘），仅保留"单纯 BMI 差异"亚组，用同样的 GLM（去掉 GDM/PE 协变量）跑一次。

**判读**：
- 两轮结果方向一致、效应量近似 → 强证据，可放心写入正文；
- 方向一致但敏感性分析显著性减弱 → 主要由样本量损失驱动，正常；
- 方向相反 → 必须警惕，主分析的"显著"很可能是合并症驱动而非 BMI 驱动，需在讨论里说明。

### 3.3 数据完整性要求

- 任一协变量缺失的样本，按"complete case"剔除，或对协变量做 MICE 多重插补（≥5 次）并 Rubin's rules 合并。
- 报告进入最终分析的样本数（CONSORT-like 流程图）。

## 阶段 4：差异代谢物挖掘与统计策略

**目标**：在小样本、多特征的现实下，平衡假阳性与假阴性，输出"可被独立验证"的候选清单。

### 4.1 弃用 OPLS-DA / VIP 的理由（保留并强化原 SOP 判断）

- 已观察到 OPLS-DA 置换检验 R²Y 与 Q² 不合格，**该模型在本数据上等同于过拟合的拟合曲线**。
- VIP 是基于该模型的特征重要性，模型本身失效则 VIP 不可信，所有基于 VIP 的筛选（如 VIP > 1）一律弃用。
- 若审稿人坚持要 PLS-DA 图，可作为"探索性可视化"附在补充材料，**不进入统计学结论链**，并在图注里写明置换检验失败。

### 4.2 主统计模型：limma / GLM

**推荐 limma（empirical Bayes 方差收缩）作为主分析模型**，理由：
- 代谢组学特征间方差差异大，普通 t/GLM 在小样本下方差估计不稳；
- limma 通过经验贝叶斯把每个特征的方差向整体收缩，**显著提升小样本下的检验功效**，且方法学已被广泛接受（源自基因表达，但对代谢组学同样有效）；
- 输出 `moderated t-statistic`、`P`、`BH-q`、`logFC`，便于直接进入下游富集分析。

**若不用 limma，退而用 GLM/ANCOVA**：
- `statsmodels` 的 OLS 即可，但记得用稳健标准误 (HC3) 防止异方差影响 P 值。

**两条路都做完后取交集** 作为"高置信"候选。

### 4.3 多重比较策略

**摒弃"原始 P<0.05 + FC>1.2 / 1.5 双限"作为唯一标准的做法**（它不控制任何 well-defined 错误率）。改用下列**三层入选规则**，逐层报告：

| 层级 | 标准 | 用途 |
|---|---|---|
| **第一层（严格）** | BH-q < 0.05 | 文章主结论引用 |
| **第二层（标准）** | BH-q < 0.10 或 Storey q < 0.10，且 \|log2FC\| ≥ log2(1.2) | 候选清单 + 后续机制讨论 |
| **第三层（探索）** | 原始 P < 0.05 且 \|log2FC\| ≥ log2(1.5) | 仅用于家族趋势观察、补充材料 |

**输出表格强制三列齐全**：`原始 P` | `BH-q (或 Storey q)` | `log2FC 及 95% CI`，让审稿人自行评估保守性。

### 4.4 离群点处理（**取消"人工肉眼剔除"**）

改成可写明的自动化规则：

1. **稳健性验证**：对每个入围特征同时跑 Wilcoxon 秩和检验，**与 limma/GLM 同时显著的优先**。
2. **离群点自动标记**：
   - 用 MAD-based 规则：`|x − median| / MAD > 3.5` 标记为离群；
   - 或 ROUT (Q = 1%) 法。
3. **不删除样本，做"敏感性结果列"**：在结果表里加一列 `is_robust_to_outlier`（剔除离群后仍显著 = TRUE）。
4. **可视化**：所有入围特征出 boxplot + dot overlay（小样本必须显示每个点）+ 协变量调整后的 partial residual plot，供读者判断。

## 阶段 5：从候选分子到生物学解释

**目标**：把分子层面的发现升级为"通路—酶—炎症网络"的可写故事。

### 5.1 家族级趋势观察（保留原 SOP 思路，方法学升级）

按上游底物与酶系归类：

| 家族 | 关注代谢物 | 生物学解读方向 |
|---|---|---|
| ARA-COX 通路 | PGE2, PGF2α, TXB2, 6-keto-PGF1α | 经典促炎与血管张力 |
| ARA-LOX 通路 | 5-/12-/15-HETE 及对应 diHETE | 5-LOX → 嗜中性招募；12/15-LOX → 巨噬极化 |
| ARA-CYP/sEH 通路 | EETs / DHETs | 血管舒张 vs sEH 高活性指向 EET 被快速水解 |
| EPA/DHA-Resolvin/Protectin/Maresin (SPMs) | RvD1/D2、PD1、MaR1、HEPE | 促消退通路；常低于 LOD，需主动报告 |
| 非酶氧化 / 自由基 | Isoprostanes（如 8-iso-PGF2α） | 氧化应激水平 |

**做法**：对每个家族绘制 heatmap（行=代谢物、列=样本、按 BMI 分组分列），叠加注释 BH-q 显著性 → 看"家族整体是否一边倒"。

### 5.2 酶活性代理比值（**强烈推荐补充**）

单分子结果常不稳，比值往往更能反映酶通量与生物学状态：

| 比值 | 反映 |
|---|---|
| `14,15-DiHET / 14,15-EET` | sEH（环氧水解酶）活性 |
| `12-HETE / AA` | 12-LOX 通量 |
| `15-HETE / AA` | 15-LOX-1 / 2 通量 |
| `PGE2 / PGD2` | COX 下游促炎 vs 消退倾斜 |
| `SPM 总和 / 促炎前列腺素总和` | "消退能力指数"（Resolution Index） |

对每个比值做与单分子相同的 GLM/limma 流程并出表。

### 5.3 正式富集 / 通路分析（**替代"家族趋势"的肉眼归纳**）

- **LION (Lipid Ontology Enrichment)**：脂质组学专用，可富集到脂质类别、亚细胞定位、生物学过程。
- **MSEA (Metabolite Set Enrichment Analysis)** via MetaboAnalyst：基于 KEGG / SMPDB 通路。
- **氧化脂质特化**：可以自建氧化脂质 panel 集合（按酶系归类的 gene set），做 GSEA-like 富集。

**输出**：富集表 + bubble plot（横轴 NES 或 enrichment ratio，纵轴 −log10 q，泡大小 = 集合内代谢物数）。

### 5.4 SPMs 检测限的主动声明

- 在结果中**列出每个 SPM 的样本检出率与组内中位数**；
- 检出率 < 50% 的 SPM 不做 GLM，仅描述检出率本身（卡方比较两组）；
- 在讨论里写明"靶向方法的 LOD 限制可能低估了消退通路的活跃程度"，避免审稿人挑战"为什么没有 RvD1"。

## 阶段 6：结果报告与可重复性

**目标**：让审稿人和重做者能完全复现你的分析，并且对结论的稳健性一目了然。

### 6.1 必出的最终表格

| 表号 | 内容 |
|---|---|
| Table 1 | 队列基线（包含 SMD 列） |
| Table 2 | 主分析差异代谢物（log2FC + 95% CI + 原始 P + BH-q + Wilcoxon P + 离群稳健性） |
| Table S1 | 完整代谢物矩阵的检出率与 CV% |
| Table S2 | 敏感性分析（纯净亚组）差异代谢物表 |
| Table S3 | 酶活性比值的统计结果 |
| Table S4 | LION / MSEA 富集结果 |

### 6.2 必出的图

1. CONSORT-style 样本流（招募 → 排除 → 最终入组 n）
2. PCA（校正前后、按年份与按分组）
3. 主分析 Volcano plot（log2FC × −log10 q）
4. 显著代谢物 Heatmap（按家族分块）
5. 关键代谢物 boxplot + 调整后 partial residual plot
6. 富集 bubble plot
7. （可选）通路示意图（手绘 ARA / EPA / DHA 网络 + 你的方向性变化箭头）

### 6.3 敏感性分析章节（在 Results 末尾）

明确列出至少三种敏感性分析的一致性：
1. 主分析（含合并症 + 协变量调整） vs 纯净亚组
2. 含离群点 vs 剔除离群点
3. 不同插补方法（半最小值 vs KNN） / 不同归一化方法（PQN vs median）
4.（可选）将 BMI 作为连续变量代替二分类

**呈现方式**：一张"一致性表"——行是代谢物，列是不同敏感性方案，单元格填方向（↑/↓）与显著性（✓/✗）。三套以上方案都一致的代谢物，写进 Discussion 主结论；不一致的写进 Limitations。

### 6.4 写作话术（强化原 SOP 的克制立场）

- **不写**："找到了 X 个 biomarker / 诊断标志物 / 用于预测 Y 的指标"。
- **写**："在调整孕周、分娩方式、孕期增重等核心临床协变量后，孕前超重/肥胖背景下胎盘 [家族 X] 氧化脂质呈现一致的趋势性升高，提示 [机制假说]，需在独立队列与机制实验中进一步验证。"
- Limitations 章节必须出现的三件事：
  1. 样本量与统计功效有限；
  2. 横断面设计无法推断因果；
  3. SPMs 等低丰度类群可能被 LOD 截断。

### 6.5 可重复性要求

- 全流程脚本归档到 Git（包含随机种子）。
- 写明每个软件包的版本号（pycombat / statsmodels / limma 等）。
- 原始数据上传至 MetaboLights 或 Metabolomics Workbench，给出 accession。
- Methods 中按 MSI（Metabolomics Standards Initiative）的 minimum reporting standards 检查清单逐项交代。

## 附录 A：常见坑速查

| 现象 | 可能原因 | 处理 |
|---|---|---|
| 校正后所有差异消失 | ComBat 与分组共线，把生物学信号一起抹了 | 回到 2.2 共线性检查；改用带 mod 的 ComBat 或直接把批次作协变量 |
| 富集到大量"脂代谢"通路，无法解读 | 通路注释过于宽泛 | 改用 LION 或自建氧化脂质 panel |
| 主分析显著，敏感性亚组不显著 | n 损失导致功效下降 / 合并症驱动 | 看效应量方向；方向一致即可放心，方向反转需重新审视 |
| 同一家族里有的升、有的降 | 异构体特异性 / 检测限差异 | 看比值；看检出率；区分 LOD 截断 vs 真实方向反转 |
| QC CV 大量 > 30% | 方法稳健性问题 / 上机条件不稳 | 与质谱平台沟通；剔除高 CV 特征 |
