## 2. CONSORT 式样本流程

```
原始样本                          n = 198
  └─ 剔除早产（GA<37 周）           −4    → 194
  └─ 剔除 BMI 缺失                  −2    → 192
  └─ 剔除体重过低（BMI<18.5）        −27   → 165
最终主分析队列                    n = 165
  ├─ 对照组：BMI 正常                120
  └─ 暴露组：超重肥胖（超重 29 + 肥胖 16）  45
```

**剔除理由**：
- 早产 (n=4)：例数太少无法分层；保留会破坏 GA 与氧化脂质的线性假设。
- BMI 缺失 (n=2)：无法归入分组。
- 体重过低 (n=27)：不在研究问题范围内（研究的是肥胖侧效应，不是消瘦）。

## 5. Table 1 绘制方案

**目的**：描述两组基线特征 + 评估均衡性，决定哪些变量必须进 GLM 协变量。

### 5.1 表格结构

- **列**：`正常 (n=120)` | `超重肥胖 (n=45)` | `Overall (n=165)` | `P` | `|SMD|`
- **行**：所有协变量按"连续 / 分类"分块罗列

### 5.2 各变量统计与检验

| 变量 | 类型 | 统计量 | 检验 |
|---|---|---|---|
| 孕妇年龄（岁） | 连续 | mean ± SD（非正态改 median (IQR)） | t / Wilcoxon |
| 孕周 `GA_decimal` | 连续 | mean ± SD | t / Wilcoxon |
| 孕期增重 GWG (kg) | 连续 | mean ± SD | t / Wilcoxon |
| 出生体重（g） | 连续 | mean ± SD | t / Wilcoxon |
| 出生体重分类 | 分类 | n (%) | χ² / Fisher |
| 胎儿性别 | 分类 | n (%) | χ² |
| GA 分类（ACOG） | 分类 | n (%) | χ² / Fisher |
| GDM | 分类 | n (%) | χ² |
| PIH（妊娠期高血压） | 分类 | n (%) | Fisher（阳性仅 8） |
| 脂代谢异常 | 分类 | n (%) | χ² |

### 5.3 默认规则

- 连续变量先 Shapiro–Wilk 检验正态性，**非正态自动改报 median (IQR) + Wilcoxon**。
- SMD 按四档阈值标记：
  - `<0.10` → ✓（可忽略）
  - `0.10–0.25` → ⚠（轻度不均衡，建议入 GLM 协变量）
  - `>0.25` → ❗（明显不均衡，必须入 GLM + 敏感性分析）
  - `>0.50` → ❗❗（严重不均衡，需在讨论中明示）
- `分娩方式` 暂不出现在 Table 1（待补全后追加）。

### 5.4 输出形式

- 主输出：Excel + 同步生成 Markdown 表（方便嵌入论文）
- 已落盘：`Table1_baseline_n165.xlsx` / `Table1_baseline_n165.md`
- 生成脚本：`generate_table1.py`

|---|---|---|---|---|---|

**检验方法**：
- 连续变量：Shapiro-Wilk 测正态性，正态用 t/Welch，非正态用 Mann-Whitney
- 2×2 分类：χ²（期望频数 ≥5）或 Fisher 精确检验
- RxC 分类（3×2）：**Fisher-Freeman-Halton 扩展（Monte Carlo 20k 模拟）**——替代之前期望频数 <5 时的 χ² 警告

### 5.6 Table 1 关键解读

1. **PIH 严重不均衡 (SMD=0.557)**：7/8 例 PIH 都在超重肥胖组。但 n=8 太少，**不能作为 GLM 协变量**；这本身是临床发现（孕前肥胖是 PIH 强危险因素），写入结果或讨论。
2. **出生体重 / 出生体重分类不均衡但属 mediator**：暴露（肥胖）→ 中介（胎儿过度生长）→ 结局，**禁止作为协变量调整**，否则会"调走"想观察的效应。
3. **孕周分类不均衡 (SMD=0.420)**：超重肥胖组早期足月比例显著更高（57.8% vs 37.5%）——临床上孕前肥胖更早择期终止。**用 `GA_decimal` 作连续协变量即可吸收**，不必再额外加分类项。
4. **年龄、孕周、GWG 三个 SMD 越线 (⚠)**：都必须进 ANCOVA。
5. **胎儿性别、GDM、脂代谢异常 SMD<0.10**：可不调整。但**GDM 出于机制相关性建议保留**作敏感性分析。

## 5.5（备注）ANCOVA / GLM / limma 三者关系

**这三个名字在本研究里指同一件事**：
- 数学上：每个代谢物拟合一个线性模型 `log2(metab) ~ BMI_group + 协变量`
- **ANCOVA**：术语强调"控制连续协变量后比较组间均值"，论文里最读者友好
- **GLM / OLS 多元线性回归**：统计学家术语
- **limma**：在 GLM/ANCOVA 基础上额外做经验贝叶斯方差收缩，小样本下功效更高

**最终方案**：主分析用 limma（实际就是 ANCOVA + 方差收缩），论文 Methods 描述用 "ANCOVA-style model with empirical Bayes variance shrinkage via the limma framework"。

## 6. 批次效应诊断（PCA + PERMANOVA）

**目的**：在跑差异分析之前，先确认 2019/2020/2021 三年送样的技术/储存差异（"年份效应"）是否强到足以扭曲生物学结论。

**输入数据**：两条数据轨各跑一次
- 主分析：`ori_n165_filtered80_log2_pareto.xlsx` (67 × 165)
- 探索性：`ori_n165_filtered50_log2_pareto.xlsx` (73 × 165)
- 用 **Pareto-scaled log2** 矩阵（而非原始 log2）——因为 PCA 对量级敏感，必须先缩放

**年份映射**：`omx_id` 前缀 → 年份
- `A19-XXX` → 2019
- `B20-XXX` → 2020
- `C21-XXX` → 2021

### 6.1 诊断方法

| 方法 | 答的问题 | 判读要点 |
|---|---|---|
| **PCA 散点（按年份/按 BMI 上色）** | 哪个因素更强地驱动样本间差异？ | 看颜色是否成簇分离 |
| **每个 PC 的 ANOVA R²** | 该 PC 的方差有多少能用年份/BMI 解释？ | R² 越大、P 越小越说明该 PC 被该因素驱动 |
| **PERMANOVA**（999 置换） | 多变量水平上，年份/BMI 是否显著解释整体方差？ | p < 0.05 即显著 |
| **年份 × BMI Cramér's V** | 年份和 BMI 是否共线？ | < 0.20 安全 / 0.20–0.50 警惕 / > 0.50 禁用 ComBat |
| **Scree plot** | 前几个 PC 是否捕获主要变异？ | 找"肘点"（elbow）；前 5 PC 应能解释 ≥ 50% |

### 6.2 如何判读 PCA 诊断图（2 × 2 子图布局）

每张诊断图含 4 个子图：

#### 子图 (0,0)：PC1 vs PC2 按年份上色 ★ 最关键
- **看年份点是否成 3 簇**：明显抱团 → 强批次效应；交错重叠 → 弱
- **看分离方向**：横向（PC1）分离意味着年份解释最大变异；纵向（PC2）则是次要变异
- **看每簇的扩散度**：紧密 = 该年内一致性高；分散 = 年内差异已经很大

#### 子图 (0,1)：PC1 vs PC2 按 BMI_group 上色 ★ 对照参考
- **期望**：在 PCA 全局上，BMI 信号通常**不应**很强地分离（因为 BMI 效应是局部、家族特异的）
- 如果 BMI 也明显分离，说明 BMI 是强驱动因子，PCA 全局水平就能看出差异，下游 ANCOVA 必然显著
- 我们的数据这里基本是混合状态——这是**正常的**，不是坏事

#### 子图 (1,0)：Scree plot 碎石图
- 蓝柱 = 单个 PC 解释方差；红线 = 累积解释方差
- **肘点**告诉你"超出第几个 PC 后边际收益就很小了"
- 我们的数据前 5 PC 累积 ~52%，**没有特别明显的肘点** → 信号比较分散（典型的代谢组学）

#### 子图 (1,1)：PC1 密度图按年份
- 三条彩色密度曲线如果**重叠** → 年份不驱动 PC1（好事）
- 如果**峰位错开** → 年份在 PC1 上有偏移（要校正）
- 比散点更直观地看一维分离

#### 决策：✅ 两条数据轨都需要做 ComBat 校正

1. 批次效应客观存在且达到 PERMANOVA 显著
2. 共线性 V=0.071，ComBat 风险低
3. 校正后预计能剥离 ~13% 的年份方差，提升下游 ANCOVA 的 BMI 信噪比

**校正参数**：
- `batch = year`（A19/B20/C21）
- `mod = BMI_group`（保护生物学变量不被一同抹掉）

### 6.6 ComBat 实施与 EB 收缩的关键调整

**实现**：Python 3.14 下 `inmoose / pycombat / combat-py` 均无可用 wheel，自实现 `scripts/utils/combat.py`（Johnson 2007 参数 EB ComBat，约 100 行）。

**首次跑发现 ComBat 几乎无效**：
- 全局方差仅变化 +0.09%
- 校正后年份 PERMANOVA F 从 3.07 **反而升到 3.30**（更显著）
- 各特征"批次间均值最大差"中位 0.362 → 0.374（**仅下降 −3.7%**）

**诊断溯源**：
- 算法本身正确，但 **EB 经验贝叶斯收缩过度**
- B20 批次（n=109，最大批次）的 `tau²=0.006`（gamma_hat 跨特征方差极小，因为其 gamma_hat 本就接近 grand mean）
- EB 后验加权 `(n·τ² + δ²)` 中分母由 δ² 主导，预期保留率仅 **40%**（< 90% 警戒线）
- 根本原因：Johnson 2007 EB 假设有**大量特征（>200）**可稳定估先验，**67 个特征不足**

**修复**：`combat(..., eb=False)`——直接用经验估计（经典 location-scale ComBat），跳过 EB 收缩。

**修复后效果**：
| 指标 | 80% 主分析 | 50% 探索性 |

### 6.8 OPLS-DA SIMCA-P 风格完整验证（含 Q²）

**为什么单独做这一节**：代谢组学审稿默认要求 **R²Y + Q² + 双置换检验**，仅报告 R²Y 不充分。`scripts/07_opls_permutation_validation.py` 提供 SIMCA-P 风格完整结果（200 次置换 × 7-fold CV）。

#### SIMCA-P 风格判定指标表

| 指标 | 80% 主分析 | 50% 探索性 | 文献阈值 | 判定 |
|---|---|---|---|---|
| R²X_pred | 0.038 | 0.036 | — | — |
| R²X_ortho | 0.186 | 0.176 | — | — |
| R²Y | 0.208 | 0.230 | >0.5 优秀 | 偏低 |
| **Q²（7-fold CV）** | **−0.222** | **−0.223** | **>0 才有意义** | ❌ **负值** |
| p_R²Y（置换） | 0.124 | 0.085 | <0.05 | ❌ 不显著 |
| **p_Q²（置换）** | **0.597** | **0.532** | <0.05 | ❌ **60% 随机置换都更好** |
| R²Y 截距（corr=0） | 0.167 | 0.182 | <0.4 | OK |
| Q² 截距（corr=0） | −0.194 | −0.206 | <0.05 | OK |

#### 关键结论

1. **Q² 为负**——根据 Westerhuis 2008 / Triba 2015 共识，`Q² < 0` 等同于"模型预测能力比仅用 y 均值 baseline 还差"，**等于无效**。
2. **p_Q² ≈ 0.6**——60% 的随机置换都能得到比实测更好的 Q²，BMI 信号完全淹没在噪声中。
3. **SIMCA-P 截距判据"看似通过"是误导**——R²Y 截距和 Q² 截距都满足阈值，**但这在小样本下会失效**。**真正的黄金标准是 Q² 实测值的正负 + p_Q²，而非截距**。
4. 论文 Methods 报告时必须同时给：`R²X_pred, R²X_ortho, R²Y, Q², p_R²Y, p_Q², R²Y intercept, Q² intercept` 全套，截距单独看会被审稿人挑战。

## 7. ANCOVA / limma 主分析结果

**目标**：在 ComBat 校正后的 log2 矩阵上，对每个氧化脂质拟合 `log2(metab) ~ BMI_group + age + GA_decimal + GWG + GDM`，提取 BMI 系数（= log2 fold change）、95% CI、原始 P 和多重比较后的 q。

### 7.1 模型与协变量选择

| 变量 | 角色 | 选入依据 |
|---|---|---|
| `BMI_group` | 暴露 | 主分析对比 (正常 0 / 超重肥胖 1) |
| `age` | 协变量 | Table 1 SMD=0.21 (⚠) |
| `GA_decimal` | 协变量 | Table 1 SMD=0.20 (⚠)；同时吸收 GA_category 的不均衡 |
| `GWG` | 协变量 | Table 1 SMD=0.15 (⚠)；5 例缺失 → complete case → n=160 |
| `GDM` | 协变量 | SMD=0.05 (√) 但机制相关，稳健性保留 |
| ~~`PIH`~~ | 不入 | SMD=0.56 (❗❗) 但阳性仅 8 例，不入模型；单独作临床发现报告 |
| ~~`出生体重` / `出生体重分类`~~ | 不入 | 中介 (暴露→中介→结局)，调整会"调走"待观察效应 |
| ~~`年份`~~ | 不入 | 已在 ComBat 阶段抹掉 |

**Complete-case 队列**：n=160 (正常 120 + 超重肥胖 40)，损失 5 例 GWG 缺失。
**残差自由度**：df_resid = 160 − 6 (intercept + 5 项) = 154。

### 7.2 三套检验并行 + EB 收缩

| 方法 | 实现 | 角色 |
|---|---|---|
| **limma EB moderated t** | 自实现 (Smyth 2004), `scripts/utils/limma.py` | **主分析**，控制小样本方差估计不稳 |
| OLS + HC3 稳健 SE | `statsmodels.OLS(cov_type='HC3')` | 防异方差；提供 log2FC 的 95% CI |
| Mann-Whitney U | `scipy.stats.mannwhitneyu` | 非参数稳健性参照 |
| MAD 离群 + 重拟合 | `|x-median|/MAD > 3.5` 标记后重跑 OLS | `is_robust_to_outlier` 列 |

**多重比较**：每套检验各做一次 BH 校正（`utils/limma.py:bh_qvalues`）。

**三层入选**（互斥，基于 limma q，与 docs §6.5/4.3 一致）：
- `strict`：q_limma < 0.05
- `standard`：q_limma < 0.10 且 |log2FC| ≥ log2(1.2)
- `exploratory`：p_limma < 0.05 且 |log2FC| ≥ log2(1.5)

**`high_confidence` 交集判据**（SOP "两条路取交集"）：
tier ∈ {strict, standard} **且** OLS p < 0.05 **且** 95% CI 不跨 0。
用于剔除 limma 在小特征数下因方差池化产生的伪阳性。

### 7.3 limma 调试：trigamma 反解 Newton 符号 bug

首版跑出 `d_0 ≈ 10¹⁴`（数值无穷），令 EB 退化为"用 s_0² 池化方差做 Z 检验"，把方差天然较大的特征（如 6,15-Diketo-PGF1α，s²=6.2）伪造成 p_limma=0.0004 的假阳性（OLS p=0.27，Wilcoxon p=0.15）。

**根因**：`_trigamma_inv` 的 Newton 迭代步用了 `y -= dy`，与 limma R 源 `y <- y + dif` 反号。`psigamma(y, deriv=2) < 0` 与分子配合的几何在 `+` 才正确。

**修复**：改为 `y += dy` + 防越界 (`y_new ≤ 0 → y/2`)。Sanity 测试：trigamma_inv(trigamma(y)) 在 y ∈ [0.5, 100] 范围内 round-trip 误差 ≤ 1e-16。

**修复后**：d_0=2.93 (80%) / 2.35 (50%)；s_0²=0.42 / 0.45，符合 67-73 特征下的轻度收缩预期。

### 7.4 双轨结果概览（修复后）

| 指标 | 80% 主轨 | 50% 探索轨 |
|---|---|---|
| 特征数 | 67 | 73 |
| 残差方差 s² 范围 | [0.116, 6.19] | [0.116, 18.00] |
| s² max/min ratio | 53.5 | 155.5 |
| log(s²) 方差 | 0.98 | 1.32 |
| limma 先验 d_0 / s_0² | 2.93 / 0.42 | 2.35 / 0.45 |
| **q_limma < 0.05** | **0** | **0** |
| **q_limma < 0.10** | **0** | **0** |
| exploratory (raw + FC) | 0 | 1 |
| **high_confidence (三认同)** | **0** | **0** |
| 离群稳健 (raw p<0.05 剔 outlier 后仍显著) | 6 / 67 | 6 / 73 |

**没有任何代谢物通过 BH-q<0.05** — 与 ComBat 校正后 PCA + PERMANOVA 给出的 **BMI 全局 p≈0.4** 完全一致；BMI 在足月胎盘氧化脂质组上的效应客观上较弱，不存在单分子级 biomarker。

### 7.5 家族级趋势（raw OLS p<0.05 且离群稳健的 6 个）

两轨高度一致，呈现**羟化 AA / DHA 衍生物族整体上行**的方向性信号：

## 10. 重要决策与依据（避免日后忘记为什么这样做）

- **剔除早产而非建模处理**：早产 n=4 太少，建模收益小于剔除；剔除后 GA 仅在 37–41+ 范围，线性假设成立。
- **超重 + 肥胖合并**：肥胖 n=16 单独无统计功效；合并后 n=45 仍小但可行；如需剂量效应可在敏感性分析里再三分类。
- **分娩方式暂不进 GLM**：缺失 18.8% 太高，强行入模型会损失过多样本；等补全后再加入并重跑。
- **OPLS-DA / VIP 全程弃用**：前期置换检验 R²Y/Q² 不合格，模型失效。
- **特征过滤选 80%+50% 双轨**：数据呈强双峰（要么 >90% 检出，要么 <50%），中间地带几乎空。80% 严格防假阳性；50% 抢救 LTB4/LTE4/8-HDHA/11-HEPE 等炎症关键分子。67% 中间档**仅多保留 2 个生物学无关特征**，弃用。依据 Bijlsma 2006 + Do 2018。
- **缺失插补走 QRILC（不走 KNN/half-min）**：氧化脂质缺失主要是 LOD 截断（MNAR）。QRILC 插补值集中在低值区（中位 0.48–0.60 vs 观测中位 15），符合生物学；若 KNN 会填出"中等丰度"反而失真。依据 Wei 2018。
- **跳过 PQN**：单位 ng/g 已是按组织湿重归一化，样本间归一化已生物学完成。代谢组学常规 PQN 主要解决尿液稀释问题，此处不必要。
- **走 ComBat 而非把年份作 ANCOVA 协变量**：年份 PERMANOVA p=0.001 显著且 PC2/PC3 解释方差较高（>8%）；Cramér's V=0.071 极低，ComBat 不会误伤生物学信号。若只把年份当协变量则只能调主效应、调不掉协方差结构，PCA 验证也更难做。
- **ComBat 关 EB 收缩 (`eb=False`)**：Johnson 2007 EB 假设 ≥200 特征才能稳定估先验，本研究 67/73 个不够，开 EB 会过度收缩到 grand mean（B20 批次预期保留率仅 40%）。改用直接经验估计后批次差下降 97.1%。这是代谢组学小特征数场景的标准应对。
- **PCA / PLS-DA / OPLS-DA 加 Hotelling's T² 95% 置信椭圆**：SIMCA-P / MetaboAnalyst 行业标配；让"组内 95% 样本预期范围"可视化，便于读者判断重叠程度。配虚线 + 半透明填充避免遮盖散点。
- **OPLS-DA 即便在批次校正后仍弃用统计推断**：置换检验 P 仍 > 0.05；只作为补充材料的视觉对照（必须配 P 值标注），不进 Results 主流程。
- **OPLS-DA 弃用最终铁证：Q² 为负 (−0.22)，p_Q² = 0.60**：模型预测能力比仅用 y 均值还差，60% 随机置换都更优；写论文时把这组指标 (R²Y/Q²/双 P/双截距) 全套放 Supplementary，配 Westerhuis 2008 / Triba 2015 文献，**主动呈现 + 主动弃用**，比"不画 OPLS"更稳妥。
- **代谢组学 OPLS-DA 必报指标 (Methods 写作)**：R²X_pred + R²X_ortho + R²Y + Q² + p_R²Y + p_Q² + R²Y intercept + Q² intercept。仅报 R²Y 或仅报截距都会被审稿挑战。**Q² 实测值的正负 + p_Q² 才是黄金标准**，截距单看会失效。
- **PCA 离群值用 T²+DModX 双重命中规则，不自动剔除**：单一指标会假阳/假阴；双重命中（严格 T²>99% AND DModX>1.5×均值）保守、可辩护。剔除前必须人工核对**技术问题 vs 生物学极端**：技术问题可剔，生物学极端要保留（可能是研究价值的发现）。剔除决定都在 Limitations 或 Supplementary 报告 + 跑敏感性分析。
- **ANCOVA 模型协变量选择 = "SMD>0.10" + "机制必入" + "无中介无共线"**：年龄/孕周/GWG 三个 SMD 越线必入；GDM 出于机制相关性保留；PIH 不入（n=8 太少）；出生体重/分类是中介（暴露→中介→结局）严禁调整；年份已在 ComBat 抹掉不再入；GA_decimal 已吸收 GA_category 不必再加分类项。**总协变量数 5 (不含截距)**，n=160 → df_resid=154 充足。
- **GWG 缺失 5 例走 complete case 而非 MICE**：缺失率仅 3%，样本损失 (165→160) 可接受；MICE 需先建立 imputation model 且与下游 limma 嵌套合并 Rubin's rules 较复杂，留作敏感性分析备选。
- **三套检验并行 (limma + OLS+HC3 + Wilcoxon)，tier 以 limma q 为主，high_confidence 用 OLS+CI 交集**：依据 SOP §4.2 "两条路都做完后取交集 作为'高置信'候选"。limma EB 在 67-73 特征 + ComBat 后方差异质 (s² max/min 53-156) 的情境下，对个别大方差特征会**过度收缩 → 伪阳性**（实测：6,15-Diketo-PGF1α p_limma=0.0004 但 p_OLS=0.27 + CI 跨 0）。`high_confidence` = limma 显著 AND OLS raw p<0.05 AND 95% CI 不跨 0，可挑出 limma 单独支撑的伪阳性。
- **自实现 limma 而非用 R / inmoose**：Python 3.14 下 R 桥接（rpy2）安装链复杂，inmoose 与 Smyth 算法等价但封装不透明，自实现 100 行 (`utils/limma.py`) + sanity test 反而最可控；同步暴露 d_0/s_0² 进审计便于学术报告。
- **limma 调试经验：trigamma 反解的 Newton 步符号必须对照 R 源码**：Smyth 2004 论文附录的算法描述容易被 Python 译者误写成 `y -= dy`（标准 Newton 直觉），但 R 源 `limma::trigammaInverse` 是 `y <- y + dif`，因为 `psigamma(y, deriv=2)<0` 与分子配合的几何关系。误写会令 d_0 → 10¹⁴（无穷大）、EB 退化为池化方差。检验方式：`trigamma_inv(trigamma(y))` 必须 round-trip 到 1e-16 精度。
- **ANCOVA 主分析 BH-q 全不显著是正确结论，不是技术失败**：与上游 PERMANOVA BMI p=0.40 完全一致；BMI 对足月胎盘氧化脂质组的影响**是局部/家族级而非全局/单分子级**。论文 Results 应坦诚呈现"单分子无 q<0.05 命中" + "家族 LOX/sEH 通路方向性趋势"，配 SOP §5.1 家族 heatmap + §5.2 酶活性比值 + §5.3 富集分析正式落地，比强行单分子级别给"假发现"更稳健。
