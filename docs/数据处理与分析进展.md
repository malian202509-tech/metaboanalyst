## 2. CONSORT 式样本流程

```
原始样本                          n = 198
  └─ 剔除早产（GA<37 周）           −4    → 194
  └─ 剔除 BMI 缺失                  −2    → 192
  └─ 剔除体重过低（BMI<18.5）        −27   → 165
最终主分析队列                    n = 165
  ├─ 对照组：BMI 正常                120
  └─ 暴露组：超重肥胖（超重 29 + 肥胖 16）  45
```

**剔除理由**：
- 早产 (n=4)：例数太少无法分层；保留会破坏 GA 与氧化脂质的线性假设。
- BMI 缺失 (n=2)：无法归入分组。
- 体重过低 (n=27)：不在研究问题范围内（研究的是肥胖侧效应，不是消瘦）。

## 5. Table 1 绘制方案

**目的**：描述两组基线特征 + 评估均衡性，决定哪些变量必须进 GLM 协变量。

### 5.1 表格结构

- **列**：`正常 (n=120)` | `超重肥胖 (n=45)` | `Overall (n=165)` | `P` | `|SMD|`
- **行**：所有协变量按"连续 / 分类"分块罗列

### 5.2 各变量统计与检验

| 变量 | 类型 | 统计量 | 检验 |
|---|---|---|---|
| 孕妇年龄（岁） | 连续 | mean ± SD（非正态改 median (IQR)） | t / Wilcoxon |
| 孕周 `GA_decimal` | 连续 | mean ± SD | t / Wilcoxon |
| 孕期增重 GWG (kg) | 连续 | mean ± SD | t / Wilcoxon |
| 出生体重（g） | 连续 | mean ± SD | t / Wilcoxon |
| 出生体重分类 | 分类 | n (%) | χ² / Fisher |
| 胎儿性别 | 分类 | n (%) | χ² |
| GA 分类（ACOG） | 分类 | n (%) | χ² / Fisher |
| GDM | 分类 | n (%) | χ² |
| PIH（妊娠期高血压） | 分类 | n (%) | Fisher（阳性仅 8） |
| 脂代谢异常 | 分类 | n (%) | χ² |

### 5.3 默认规则

- 连续变量先 Shapiro–Wilk 检验正态性，**非正态自动改报 median (IQR) + Wilcoxon**。
- SMD 按四档阈值标记：
  - `<0.10` → ✓（可忽略）
  - `0.10–0.25` → ⚠（轻度不均衡，建议入 GLM 协变量）
  - `>0.25` → ❗（明显不均衡，必须入 GLM + 敏感性分析）
  - `>0.50` → ❗❗（严重不均衡，需在讨论中明示）
- `分娩方式` 暂不出现在 Table 1（待补全后追加）。

### 5.4 输出形式

- 主输出：Excel + 同步生成 Markdown 表（方便嵌入论文）
- 已落盘：`Table1_baseline_n165.xlsx` / `Table1_baseline_n165.md`
- 生成脚本：`generate_table1.py`

### 5.5 Table 1 实际结果（n=165）

| 变量 | 正常 (n=120) | 超重肥胖 (n=45) | P | \|SMD\| | 提示 |
|---|---|---|---|---|---|
| 孕妇年龄 (岁) | 31.00 (29, 35) | 31.00 (29, 35) | 0.483 | 0.212 | ⚠ |
| 孕周 (周) | 39.00 (38.71, 39.57) | 38.71 (38.43, 39.43) | 0.074 | 0.200 | ⚠ |
| 孕期增重 (kg) | 13.40 ± 4.48 | 12.70 ± 5.10 | 0.405 | 0.147 | ⚠ |
| 出生体重 (g) | 3419 ± 368 | 3579 ± 419 | 0.018 | 0.405 | ! |
| 胎儿性别 (男%) | 51.7% | 51.1% | 1.000 | 0.011 | √ |
| 孕周分类 (ACOG) | 早期足月 37.5% | 早期足月 57.8% | 0.088 | 0.420 | ! |
| 出生体重分类 (巨大儿%) | 5.8% | 17.8% | 0.056 | 0.397 | ! |
| GDM (%) | 15.0% | 13.3% | 0.982 | 0.048 | √ |
| PIH (%) | 0.8% | 15.6% | <0.001 | 0.557 | ❗❗ |
| 脂代谢异常 (%) | 30.8% | 33.3% | 0.905 | 0.054 | √ |

**检验方法**：
- 连续变量：Shapiro-Wilk 测正态性，正态用 t/Welch，非正态用 Mann-Whitney
- 2×2 分类：χ²（期望频数 ≥5）或 Fisher 精确检验
- RxC 分类（3×2）：**Fisher-Freeman-Halton 扩展（Monte Carlo 20k 模拟）**——替代之前期望频数 <5 时的 χ² 警告

### 5.6 Table 1 关键解读

1. **PIH 严重不均衡 (SMD=0.557)**：7/8 例 PIH 都在超重肥胖组。但 n=8 太少，**不能作为 GLM 协变量**；这本身是临床发现（孕前肥胖是 PIH 强危险因素），写入结果或讨论。
2. **出生体重 / 出生体重分类不均衡但属 mediator**：暴露（肥胖）→ 中介（胎儿过度生长）→ 结局，**禁止作为协变量调整**，否则会"调走"想观察的效应。
3. **孕周分类不均衡 (SMD=0.420)**：超重肥胖组早期足月比例显著更高（57.8% vs 37.5%）——临床上孕前肥胖更早择期终止。**用 `GA_decimal` 作连续协变量即可吸收**，不必再额外加分类项。
4. **年龄、孕周、GWG 三个 SMD 越线 (⚠)**：都必须进 ANCOVA。
5. **胎儿性别、GDM、脂代谢异常 SMD<0.10**：可不调整。但**GDM 出于机制相关性建议保留**作敏感性分析。

## 5.5（备注）ANCOVA / GLM / limma 三者关系

**这三个名字在本研究里指同一件事**：
- 数学上：每个代谢物拟合一个线性模型 `log2(metab) ~ BMI_group + 协变量`
- **ANCOVA**：术语强调"控制连续协变量后比较组间均值"，论文里最读者友好
- **GLM / OLS 多元线性回归**：统计学家术语
- **limma**：在 GLM/ANCOVA 基础上额外做经验贝叶斯方差收缩，小样本下功效更高

**最终方案**：主分析用 limma（实际就是 ANCOVA + 方差收缩），论文 Methods 描述用 "ANCOVA-style model with empirical Bayes variance shrinkage via the limma framework"。

## 6. 批次效应诊断（PCA + PERMANOVA）

**目的**：在跑差异分析之前，先确认 2019/2020/2021 三年送样的技术/储存差异（"年份效应"）是否强到足以扭曲生物学结论。

**输入数据**：两条数据轨各跑一次
- 主分析：`ori_n165_filtered80_log2_pareto.xlsx` (67 × 165)
- 探索性：`ori_n165_filtered50_log2_pareto.xlsx` (73 × 165)
- 用 **Pareto-scaled log2** 矩阵（而非原始 log2）——因为 PCA 对量级敏感，必须先缩放

**年份映射**：`omx_id` 前缀 → 年份
- `A19-XXX` → 2019
- `B20-XXX` → 2020
- `C21-XXX` → 2021

### 6.1 诊断方法

| 方法 | 答的问题 | 判读要点 |
|---|---|---|
| **PCA 散点（按年份/按 BMI 上色）** | 哪个因素更强地驱动样本间差异？ | 看颜色是否成簇分离 |
| **每个 PC 的 ANOVA R²** | 该 PC 的方差有多少能用年份/BMI 解释？ | R² 越大、P 越小越说明该 PC 被该因素驱动 |
| **PERMANOVA**（999 置换） | 多变量水平上，年份/BMI 是否显著解释整体方差？ | p < 0.05 即显著 |
| **年份 × BMI Cramér's V** | 年份和 BMI 是否共线？ | < 0.20 安全 / 0.20–0.50 警惕 / > 0.50 禁用 ComBat |
| **Scree plot** | 前几个 PC 是否捕获主要变异？ | 找"肘点"（elbow）；前 5 PC 应能解释 ≥ 50% |

### 6.2 如何判读 PCA 诊断图（2 × 2 子图布局）

每张诊断图含 4 个子图：

#### 子图 (0,0)：PC1 vs PC2 按年份上色 ★ 最关键
- **看年份点是否成 3 簇**：明显抱团 → 强批次效应；交错重叠 → 弱

### 6.6 ComBat 实施与 EB 收缩的关键调整

**实现**：Python 3.14 下 `inmoose / pycombat / combat-py` 均无可用 wheel，自实现 `scripts/utils/combat.py`（Johnson 2007 参数 EB ComBat，约 100 行）。

**首次跑发现 ComBat 几乎无效**：
- 全局方差仅变化 +0.09%
- 校正后年份 PERMANOVA F 从 3.07 **反而升到 3.30**（更显著）
- 各特征"批次间均值最大差"中位 0.362 → 0.374（**仅下降 −3.7%**）

**诊断溯源**：
- 算法本身正确，但 **EB 经验贝叶斯收缩过度**
- B20 批次（n=109，最大批次）的 `tau²=0.006`（gamma_hat 跨特征方差极小，因为其 gamma_hat 本就接近 grand mean）
- EB 后验加权 `(n·τ² + δ²)` 中分母由 δ² 主导，预期保留率仅 **40%**（< 90% 警戒线）
- 根本原因：Johnson 2007 EB 假设有**大量特征（>200）**可稳定估先验，**67 个特征不足**

**修复**：`combat(..., eb=False)`——直接用经验估计（经典 location-scale ComBat），跳过 EB 收缩。

**修复后效果**：
| 指标 | 80% 主分析 | 50% 探索性 |
|---|---|---|
| 校正前年份 PERMANOVA F | 3.07 | 3.04 |
| **校正后年份 PERMANOVA F** | **0.003** | **0.003** |
| **校正后年份 p** | **1.000** | **1.000** |
| 批次均值最大差下降 | **97.1%** | 类似 |
| 校正后 BMI PERMANOVA p | 0.40 | 0.35 |

**结论**：年份效应被完全剥离，BMI 信号未受影响（共线性低 V=0.071，符合预期）。

### 6.8 OPLS-DA SIMCA-P 风格完整验证（含 Q²）

**为什么单独做这一节**：代谢组学审稿默认要求 **R²Y + Q² + 双置换检验**，仅报告 R²Y 不充分。`scripts/07_opls_permutation_validation.py` 提供 SIMCA-P 风格完整结果（200 次置换 × 7-fold CV）。

#### SIMCA-P 风格判定指标表

| 指标 | 80% 主分析 | 50% 探索性 | 文献阈值 | 判定 |
|---|---|---|---|---|
| R²X_pred | 0.038 | 0.036 | — | — |
| R²X_ortho | 0.186 | 0.176 | — | — |
| R²Y | 0.208 | 0.230 | >0.5 优秀 | 偏低 |
| **Q²（7-fold CV）** | **−0.222** | **−0.223** | **>0 才有意义** | ❌ **负值** |
| p_R²Y（置换） | 0.124 | 0.085 | <0.05 | ❌ 不显著 |
| **p_Q²（置换）** | **0.597** | **0.532** | <0.05 | ❌ **60% 随机置换都更好** |
| R²Y 截距（corr=0） | 0.167 | 0.182 | <0.4 | OK |
| Q² 截距（corr=0） | −0.194 | −0.206 | <0.05 | OK |

#### 关键结论

1. **Q² 为负**——根据 Westerhuis 2008 / Triba 2015 共识，`Q² < 0` 等同于"模型预测能力比仅用 y 均值 baseline 还差"，**等于无效**。
2. **p_Q² ≈ 0.6**——60% 的随机置换都能得到比实测更好的 Q²，BMI 信号完全淹没在噪声中。
3. **SIMCA-P 截距判据"看似通过"是误导**——R²Y 截距和 Q² 截距都满足阈值，**但这在小样本下会失效**。**真正的黄金标准是 Q² 实测值的正负 + p_Q²，而非截距**。
4. 论文 Methods 报告时必须同时给：`R²X_pred, R²X_ortho, R²Y, Q², p_R²Y, p_Q², R²Y intercept, Q² intercept` 全套，截距单独看会被审稿人挑战。

### 7.2 三套检验并行 + EB 收缩

| 方法 | 实现 | 角色 |
|---|---|---|
| **limma EB moderated t** | 自实现 (Smyth 2004), `scripts/utils/limma.py` | **主分析**，控制小样本方差估计不稳 |
| OLS + HC3 稳健 SE | `statsmodels.OLS(cov_type='HC3')` | 防异方差；提供 log2FC 的 95% CI |
| Mann-Whitney U | `scipy.stats.mannwhitneyu` | 非参数稳健性参照 |
| **OLS internal studentized residual** 离群 + 重拟合 | `|r_int| > 3` 标记后重跑 OLS+HC3 | **多列稳健性指标**（见 §7.7 评论修复） |

**多重比较**：每套检验各做一次 BH 校正（`utils/limma.py:bh_qvalues`）。

**三层入选**（互斥，基于 limma q，与 docs §6.5/4.3 一致）：
- `strict`：q_limma < 0.05
- `standard`：q_limma < 0.10 且 |log2FC| ≥ log2(1.2)
- `exploratory`：p_limma < 0.05 且 |log2FC| ≥ log2(1.5)

**`high_confidence` 交集判据**（SOP "两条路取交集"）：
tier ∈ {strict, standard} **且** OLS p < 0.05 **且** 95% CI 不跨 0。
用于剔除 limma 在小特征数下因方差池化产生的伪阳性。

**`is_robust_to_outlier` 多列判据**（新版，2026-05-18 重写）：
- `n_outliers = 0` → 自动 True（无离群无需检验）
- `n_outliers > 0` → 必须同时满足：方向一致 (`direction_consistent`) AND 效应衰减 ≤ 50% (`|attenuation_pct| ≤ 0.5`) AND `p_robust_ols < 0.05`
- 旧版仅看 `p_robust < 0.05`，会把"方向反转"或"效应几乎消失"的伪稳健也判为 True。新版同步输出 `beta_after`/`attenuation_pct` 供独立核对。

### 7.3 limma 调试：trigamma 反解 Newton 符号 bug

首版跑出 `d_0 ≈ 10¹⁴`（数值无穷），令 EB 退化为"用 s_0² 池化方差做 Z 检验"，把方差天然较大的特征（如 6,15-Diketo-PGF1α，s²=6.2）伪造成 p_limma=0.0004 的假阳性（OLS p=0.27，Wilcoxon p=0.15）。

**根因**：`_trigamma_inv` 的 Newton 迭代步用了 `y -= dy`，与 limma R 源 `y <- y + dif` 反号。`psigamma(y, deriv=2) < 0` 与分子配合的几何在 `+` 才正确。

**修复**：改为 `y += dy` + 防越界 (`y_new ≤ 0 → y/2`)。Sanity 测试：trigamma_inv(trigamma(y)) 在 y ∈ [0.5, 100] 范围内 round-trip 误差 ≤ 1e-16。

**修复后**：d_0=2.93 (80%) / 2.35 (50%)；s_0²=0.42 / 0.45，符合 67-73 特征下的轻度收缩预期。

### 7B.5b DHA 通路最终图景与 16-HDoHE 重归属决策 (2026-05-19)

#### 16-HDoHE 重归属：从 CYP-ω 改为 Non-enzymatic (radical)

**文献依据**：
- VanRollins 2008 *J Lipid Res* — CYP4-DHA 主产物是 19/20/22-HDHA，ω-7 位 (C16) 不是 CYP4 偏好位置
- Yin & Porter 2005 *Antioxid Redox Signal* — DHA 自由基自氧化经 C15 双烯丙位 H 脱氢 → C16 radical addition → 16-OH + 17E 双键重排（panel 命名 "16-Hydroxy-...,17E,19Z-" 完全契合该机制）

**数据实证（9 HDoHE 单分子 ANCOVA, n=165, scripts/15c_16hdohe_origin_check.py）**：

| 分子 | 假设来源 | log2FC | p_OLS | p_Wilc | 聚类 |
|---|---|---|---|---|---|
| **20-HDoHE** | **CYP4 纯** | **+0.333** | **0.022 ✓** | **0.019 ✓** | CYP 顶部 |
| 14-HDoHE | 12-LOX | +0.269 | 0.131 | 0.133 | LOX 中位 |
| 11-HDoHE | 非酶 | +0.217 | 0.289 | 0.021 | 非酶上沿 |
| **16-HDoHE** | **争议** | **+0.173** | 0.189 | 0.128 | **非酶中位** |
| 13-HDoHE | 非酶 | +0.170 | 0.059 | 0.057 | 非酶中位 |
| 17-HDoHE | 15-LOX | +0.155 | 0.312 | 0.237 | 弱 LOX |
| 7/10-HDoHE | 非酶 | +0.124~+0.133 | ~0.32 | ~0.20 | 非酶下沿 |
| 4-HDoHE | 5-LOX+非酶 | +0.001 | 0.997 | 0.589 | unchanged |

**16-HDoHE log2FC (+0.173) 与非酶组中位 (+0.170) 几乎相同，远低于纯 CYP 20-HDoHE (+0.333)**。

**比值层验证**：
- (16+20)/DHA 旧混合版: log2FC=+0.393, FC=1.31×, p=0.009
- **20-HDoHE/DHA 单独 (剔除 16)**: log2FC=+0.429, FC=**1.35×**, p=**0.008** ← **信号反而增强**
- 加 16 到自氧化指数 (4+7+10+11+13+16)/DHA vs 不加 (4+7+10+11+13)/DHA 几乎无差异 (log2FC 0.281 → 0.281)
- **结论**：16-HDoHE 的统计行为完全融入非酶组，作为 CYP 产物会稀释信号 → 归非酶。

#### DHA 浓度组间对比的关键结论

1. **DHA 单分子层面没有显著差异** (p_OLS=0.172, p_Wilcoxon=0.507, CI 跨 0). 中位浓度从 14627 → 约 13750 ng/g, 下降 ≈6%.
2. **但方向稳定且与所有 PUFA 一致**——7 个独立测量的 PUFA 都 ↓ 6-13%, log2FC 几乎相同 (SD 极小), **这不是噪声**, 是真实的"全 PUFA 池广谱消耗"现象.
3. **统计功效不足是单分子不显著的主因**: 6% 浓度差 + n=120 vs 45 不平衡设计下 Cohen's d≈0.18, 80% power 需 n≈350/组. 这是**生理上真实但样本量限制下达不到 p<0.05** 的典型场景.
4. **GSEA 家族级强信号 (q=0.004) 恰好说明问题**: 5 个 PUFA 各自 p≈0.15 不显著, 但**方向一致 + 排名靠后**给出强家族富集——这是 GSEA 的经典优势体现"small consistent effects accumulate".

#### DHA 不显著 ↓ 的临床/机制意义

| 维度 | 临床含义 |
|---|---|
| **胎儿神经发育** | DHA 经胎盘主动转运给胎儿, 是脑+视网膜发育核心营养; 母体胎盘 DHA 池 ↓ 6% → 胎儿可获得性 DHA 减少风险 |

| **跨代代谢编程** | 胎盘 DHA 储备不足与子代神经认知 + 代谢综合征风险关联 (Innis 2011 *Placenta*; Calder 2016 *Biochimie*) |

#### LA 池关键发现

1. **CYP-Epo (EpOME/LA) p=0.040 ★ — LA 池唯一显著通路**. EpOME 是 **leukotoxin 前体** (经 sEH → DiHOME = leukotoxin diol, 与全身炎症综合征 / 急性肺损伤关联). 同时 LA 池 sEH 趋势 ↓ (p=0.22) → **EpOME 累积而 DiHOME 水解滞后** → 胎盘 leukotoxin 前体潜在累积. 这是 LA 池在孕前肥胖背景下的核心机制信号.
2. **LA-LOX 通路 (13-HODE/LA) p=0.42 几乎不动** — 与 AA 池 12/15-LOX (p≈0.09 接近显著) 形成对比, **胎盘 LOX 酶系底物偏好 AA >> LA**. 文献支持: ALOX12/ALOX15 在胎盘对 AA 的 Km 显著低于 LA.
3. **LA 池非酶氧化 (9-HODE/LA) 完全不动** (p=0.523) — 这是与 AA / DHA 池非酶氧化显著 ↑ 形成最有信息量的对比, 详见下方三池对比.
4. **oxoODE 失活通路也完全不动** (p=0.811) — LA 池整体氧化通量在孕前肥胖背景下变化轻微, 只有 CYP-Epo 这一条酶通路被激活.

#### **三大底物池非酶氧化通路对比 — PUFA 双键数梯度效应**

这是本研究**最有理论价值的发现之一**:

| 底物池 | 双键数 | 双烯丙位数 | 非酶氧化比值 | log2FC | p_OLS | 显著 |
|---|---|---|---|---|---|---|
| **DHA 池** (22:6) | **6** | **5** | (7+10+11+13+16-HDoHE)/DHA | **+0.314** | **0.012 ✅** | ✓ |
| **AA 池** (20:4) | **4** | **3** | (11+8-HETE)/AA | **+0.223** | **0.032 ✅** | ✓ |
| **LA 池** (18:2) | **2** | **1** | 9-HODE/LA | +0.062 | 0.523 | ❌ |

**理论支撑**: Yin & Porter 2005 *Antioxid Redox Signal* + Niki 2009 *Free Radic Biol Med* 已证明 **PUFA 自由基自氧化速率与可氧化位点 (双烯丙位 H) 数量呈指数级相关** (rate ∝ exp(n_bisallylic_H)). 我们的数据完美复现了这个机制:
- DHA (5 双烯丙位) → 自由基氧化速率最快 → 信号最强 (log2FC +0.31, p=0.012)
- AA (3 双烯丙位) → 次之 (log2FC +0.22, p=0.032)
- LA (1 双烯丙位) → 抗氧化, 几乎不响应 (log2FC +0.06, p=0.52)

**这一梯度的方法学价值**: 反向证明 AA + DHA 非酶氧化 ↑ **不是技术伪影或基线漂移**——如果是, 所有 PUFA 都该同方向变化. **LA 不动 + AA/DHA 显著 ↑ 的对比图景 = 真实生物学的内部对照**, 远比单一底物的发现更可靠.

#### **整体生物学叙述 (新版本, 论文 Discussion 候选)**

> 孕前肥胖在胎盘上对三大主底物 PUFA 池 (AA / LA / DHA) 介导了**底物特异性**的氧化代谢重塑:
>
> - **DHA 池** (22:6, 5 双烯丙位) — 4 条通路同时激活 (CYP4-ω q=0.008 / LOX SPM 前体 p=0.048 / 非酶自氧化 p=0.012), 5-LOX 独立 unchanged. 母体浓度同方向 ↓6% (家族 GSEA q=0.004).
> - **AA 池** (20:4, 3 双烯丙位) — COX/LOX 双酶系并行激活 (3 条接近显著), 非酶自氧化 p=0.032 显著, CYP4-ω 反向 ↓ (被 DHA 抢占), 内源大麻素 AEA ↓.
> - **LA 池** (18:2, 1 双烯丙位) — **CYP-Epo (EpOME → leukotoxin 前体累积) 唯一显著激活 (p=0.040)**, LOX 和非酶氧化都未达显著, 这是 PUFA 双键少导致的内在氧化抗性.
>
> **三池非酶氧化通路按"PUFA 双键数梯度"激活** (DHA p=0.012 / AA p=0.032 / LA p=0.523), 与 Yin & Porter 2005 自由基氧化速率与双烯丙位数指数相关的经典机制完美吻合. 这一梯度反过来排除了"全 PUFA 池无差别氧化"的技术伪影解释, 证实孕前肥胖介导的胎盘氧化应激是**选择性攻击高度不饱和 PUFA**的真实生物学过程, 且**酶系底物偏好显著重塑** (CYP4 偏 DHA / LOX 偏 AA / CYP-Epo 偏 LA), 形成跨底物的代谢酶系分流重塑图景.

## 10. 重要决策与依据（避免日后忘记为什么这样做）

- **剔除早产而非建模处理**：早产 n=4 太少，建模收益小于剔除；剔除后 GA 仅在 37–41+ 范围，线性假设成立。
- **超重 + 肥胖合并**：肥胖 n=16 单独无统计功效；合并后 n=45 仍小但可行；如需剂量效应可在敏感性分析里再三分类。
- **分娩方式暂不进 GLM**：缺失 18.8% 太高，强行入模型会损失过多样本；等补全后再加入并重跑。
- **OPLS-DA / VIP 全程弃用**：前期置换检验 R²Y/Q² 不合格，模型失效。
- **特征过滤选 80%+50% 双轨**：数据呈强双峰（要么 >90% 检出，要么 <50%），中间地带几乎空。80% 严格防假阳性；50% 抢救 LTB4/LTE4/8-HDHA/11-HEPE 等炎症关键分子。67% 中间档**仅多保留 2 个生物学无关特征**，弃用。依据 Bijlsma 2006 + Do 2018。
- **缺失插补走 QRILC（不走 KNN/half-min）**：氧化脂质缺失主要是 LOD 截断（MNAR）。QRILC 插补值集中在低值区（中位 0.48–0.60 vs 观测中位 15），符合生物学；若 KNN 会填出"中等丰度"反而失真。依据 Wei 2018。
- **跳过 PQN**：单位 ng/g 已是按组织湿重归一化，样本间归一化已生物学完成。代谢组学常规 PQN 主要解决尿液稀释问题，此处不必要。
- **走 ComBat 而非把年份作 ANCOVA 协变量**：年份 PERMANOVA p=0.001 显著且 PC2/PC3 解释方差较高（>8%）；Cramér's V=0.071 极低，ComBat 不会误伤生物学信号。若只把年份当协变量则只能调主效应、调不掉协方差结构，PCA 验证也更难做。
- **ComBat 关 EB 收缩 (`eb=False`)**：Johnson 2007 EB 假设 ≥200 特征才能稳定估先验，本研究 67/73 个不够，开 EB 会过度收缩到 grand mean（B20 批次预期保留率仅 40%）。改用直接经验估计后批次差下降 97.1%。这是代谢组学小特征数场景的标准应对。
- **PCA / PLS-DA / OPLS-DA 加 Hotelling's T² 95% 置信椭圆**：SIMCA-P / MetaboAnalyst 行业标配；让"组内 95% 样本预期范围"可视化，便于读者判断重叠程度。配虚线 + 半透明填充避免遮盖散点。
- **OPLS-DA 即便在批次校正后仍弃用统计推断**：置换检验 P 仍 > 0.05；只作为补充材料的视觉对照（必须配 P 值标注），不进 Results 主流程。
- **OPLS-DA 弃用最终铁证：Q² 为负 (−0.22)，p_Q² = 0.60**：模型预测能力比仅用 y 均值还差，60% 随机置换都更优；写论文时把这组指标 (R²Y/Q²/双 P/双截距) 全套放 Supplementary，配 Westerhuis 2008 / Triba 2015 文献，**主动呈现 + 主动弃用**，比"不画 OPLS"更稳妥。
- **代谢组学 OPLS-DA 必报指标 (Methods 写作)**：R²X_pred + R²X_ortho + R²Y + Q² + p_R²Y + p_Q² + R²Y intercept + Q² intercept。仅报 R²Y 或仅报截距都会被审稿挑战。**Q² 实测值的正负 + p_Q² 才是黄金标准**，截距单看会失效。
- **PCA 离群值用 T²+DModX 双重命中规则，不自动剔除**：单一指标会假阳/假阴；双重命中（严格 T²>99% AND DModX>1.5×均值）保守、可辩护。剔除前必须人工核对**技术问题 vs 生物学极端**：技术问题可剔，生物学极端要保留（可能是研究价值的发现）。剔除决定都在 Limitations 或 Supplementary 报告 + 跑敏感性分析。
- **ANCOVA 模型协变量选择 = "SMD>0.10" + "机制必入" + "无中介无共线"**：年龄/孕周/GWG 三个 SMD 越线必入；GDM 出于机制相关性保留；PIH 不入（n=8 太少）；出生体重/分类是中介（暴露→中介→结局）严禁调整；年份已在 ComBat 抹掉不再入；GA_decimal 已吸收 GA_category 不必再加分类项。**总协变量数 5 (不含截距)**，n=160 → df_resid=154 充足。
- **GWG 缺失 5 例走 complete case 而非 MICE**：缺失率仅 3%，样本损失 (165→160) 可接受；MICE 需先建立 imputation model 且与下游 limma 嵌套合并 Rubin's rules 较复杂，留作敏感性分析备选。
- **三套检验并行 (limma + OLS+HC3 + Wilcoxon)，tier 以 limma q 为主，high_confidence 用 OLS+CI 交集**：依据 SOP §4.2 "两条路都做完后取交集 作为'高置信'候选"。limma EB 在 67-73 特征 + ComBat 后方差异质 (s² max/min 53-156) 的情境下，对个别大方差特征会**过度收缩 → 伪阳性**（实测：6,15-Diketo-PGF1α p_limma=0.0004 但 p_OLS=0.27 + CI 跨 0）。`high_confidence` = limma 显著 AND OLS raw p<0.05 AND 95% CI 不跨 0，可挑出 limma 单独支撑的伪阳性。
- **自实现 limma 而非用 R / inmoose**：Python 3.14 下 R 桥接（rpy2）安装链复杂，inmoose 与 Smyth 算法等价但封装不透明，自实现 100 行 (`utils/limma.py`) + sanity test 反而最可控；同步暴露 d_0/s_0² 进审计便于学术报告。
- **limma 调试经验：trigamma 反解的 Newton 步符号必须对照 R 源码**：Smyth 2004 论文附录的算法描述容易被 Python 译者误写成 `y -= dy`（标准 Newton 直觉），但 R 源 `limma::trigammaInverse` 是 `y <- y + dif`，因为 `psigamma(y, deriv=2)<0` 与分子配合的几何关系。误写会令 d_0 → 10¹⁴（无穷大）、EB 退化为池化方差。检验方式：`trigamma_inv(trigamma(y))` 必须 round-trip 到 1e-16 精度。
- **ANCOVA 主分析 BH-q 全不显著是正确结论，不是技术失败**：与上游 PERMANOVA BMI p=0.40 完全一致；BMI 对足月胎盘氧化脂质组的影响**是局部/家族级而非全局/单分子级**。论文 Results 应坦诚呈现"单分子无 q<0.05 命中" + "家族 LOX/sEH 通路方向性趋势"，配 SOP §5.1 家族 heatmap + §5.2 酶活性比值 + §5.3 富集分析正式落地，比强行单分子级别给"假发现"更稳健。
- **方案 D（ComBat by batch + year 作协变量），不走方案 A/B/C**：拿到真实上机批次表后发现 batch（2 个）与 year（3 个）几乎独立（Cramér's V=0.074）、各自在 PCA 不同 PC 主导，**不能互替**。方案 A（顺序两次 ComBat）顺序敏感会过校正、方案 B（合并 6 组）最小组 6 例方差不稳、方案 C（双 batch 模型自实现）开发成本高且非 EB。方案 D 把"技术变异（仪器/试剂批/色谱柱状态切换）"和"年代/季节/储存复合效应"分开处理 — batch 用 ComBat 抹（属纯技术），year 用 ANCOVA 协变量调（可能含部分生物学信号，强抹有误伤风险）。代价：ANCOVA 多 2 个 df，n=160 → df_resid 154→152，完全负担得起。
- **首次 ComBat 用 year 当 batch 是误用且需诚实记录**：当时无上机批次表，用 omx_id 年份前缀 (A19/B20/C21) 作 batch 代理，技术上合理。但后续诊断证明 batch 与 year 各自独立显著，"用 year 代理 batch"实际上**只校正了 year 不能也不该校正的部分** + **完全没碰真正的 batch 信号**。教训：**有疑义时先要供应商的上机顺序与批次表**，不要用样本 ID 推断。这是观察性研究质控的常见盲区，文档全程记录该过程作为方法论参考。
- **协变量经病历补齐 + Table 1 重测后二轮删减（2026-05-18）**：分娩方式实际 SMD=0.036 √ 远低 0.10 阈值（病历补齐前估计为强混杂，补齐后证实不均衡极小）；GWG 是 mediator-like、GDM SMD=0.05、产次未采集；按 Austin SMD 规则 + mediator 排除规则，删 4 留 3（age + GA + year）。**结论：补齐病历后基于实测数据重做协变量决策是正确流程，避免基于"先验担心"过度调整**。删 GWG 副作用：恰好解决了外部代码评审意见 #5（mediator 估的是直接效应而非总效应）；删 GDM 让模型更简洁；保留 age/GA/year 是 SMD 越线 + 方案 D 切换后 year 必入的硬约束。
- **离群检测 MAD→studentized residual（2026-05-18 评审采纳）**：旧版 `|y-median|/MAD>3.5` 对原始 y 做检测，在 ANCOVA 场景下会把分布偏态或组间结构当离群（实测：旧版 50 轨某特征 n_outliers=31，相当于 19% 样本被误判）；新版改 OLS internal studentized residual `|r_int|>3`，残差去除协变量与组间效应后再判，n=165 下理论期望约 0.4 个/特征，实测最大 7-8、中位 2，与理论吻合。**ANCOVA 场景下基于残差判离群是黄金标准**，不要再写 MAD-on-y。
- **稳健性判据多列化（2026-05-18 评审采纳）**：旧版 `is_robust_to_outlier = (p_robust<0.05)` 单列判据会把"剔后方向反转"或"效应几乎全部消失"也判 True；新版必须同时满足 `direction_consistent` AND `|attenuation_pct|≤0.5` AND `p_robust<0.05`，n_outliers=0 时直接 True。同时保留 `beta_after`/`attenuation_pct` 等中间列供独立核对。这才是"稳健性"的正确语义。
- **80%/50% 双轨明确为主轨 + 补救/敏感性（2026-05-18 评审采纳）**：50 轨 73 个特征中 67 个就是 80 轨子集（重叠 92%），多出的 6 个是 LTB4/LTE4/11-HEPE/9-HpODE/8-HDoHE/13,14-Dihydro-15-keto-PGE1（炎症关键分子，验证了 §10 之前"50% 抢救炎症分子"决策）。**不能当独立发现集**：50 轨结果加 `also_in_80_main` 列显式标记，论文按"80 为主分析、50 为补救/敏感性"叙述，不重复计数。
- **下游分析（候选/heatmap/boxplot/酶比值/富集）一律以 80 主轨为主结论（2026-05-18 用户锁定）**：通路机制实证对比 (§7B.6) 显示 80 主轨给出 5 个 BH-q<0.05 显著家族（AA-LOX↑/EPA-DHA-DPA↑/AA-CYP-sEH↓/Free PUFA↓↓/Endocannabinoid↓）vs 50 探索轨仅 1 个 (Free PUFA q=0.004)。原因：50 轨多的 6 个低检出特征虽方向也一致但 mean_t 较弱，**稀释家族整体效应**（如 AA-LOX 从 mean_t=+1.21→+1.01）。**50 轨价值在于"7/7 家族方向 100% 一致"的稳健性证据**，而非补 5 个独立显著家族。论文 Results 一段叙述 80 主轨；Supplementary 放 50 轨富集表 + 一句话 "all 7 families showed concordant direction at relaxed 50% threshold, confirming robustness"。
- **差异候选筛选标准 = `p_limma<0.05 AND |log2FC|≥log2(1.2)` 且离群稳健（2026-05-18 用户锁定）**：介于 SOP §4.3 standard (q<0.10+FC≥1.2) 和 exploratory (p<0.05+FC≥1.5) 之间, **不控制 FDR**。Methods 必须明示为 "hypothesis-generating tier"，不能写 "differentially abundant metabolites"。每个候选另由 Wilcoxon + 离群稳健性互校。**80 主轨 H0 假阳性期望 3.35 vs 实际命中 2** = 低于期望, 增强可信度。
- **下游富集首选 GSEA-like 家族而非 KEGG（2026-05-18 决定）**：KEGG 评估发现两条致命问题: (1) 80 主轨候选 2 个 KEGG ID 覆盖 0/2，50 轨 4 个仅 1/4 → 候选无法映射, ORA 在 KEGG 上几乎全空; (2) 氧化脂质 panel ~85% 集中 hsa00590 一条通路, KEGG 区分度天然差。改用我们 SOP §5.1 自定义 7 大家族（按酶系归类: AA-COX/LOX/CYP-sEH/EPA-DHA-DPA/LA/Endocannabinoid/Free PUFA）做 GSEA-like, 得到 5/7 BH-q<0.05 显著结果。ORA 作为诊断辅助（脚本 17）展示候选在 3 维度（family/substrate/enzyme）的命中, 但 BH-q 在候选 n=2-4 下无法显著, 是数学限制非数据问题。
- **酶活性比值用 "log2 反 log → 同族求和 → log2 → ANCOVA" 而非 log 尺度直接均值差**：严格保留 raw 浓度的求和语义, 避免几何均值简化导致与"家族总通量"语义偏离。代价: ComBat 在 log2 尺度做的校正反 log2 求和后理论上有轻微偏差, 但实践标准做法 (Wei 2018, MetaboAnalystR 同方法)。
- **HETrE (mono-hydroxy ETrE) 归 AA-LOX 而非 AA-CYP/sEH（2026-05-18 分类细节修正）**：化学上 12-HETrE 与 12-HETE 同源酶系 (LOX → HpETrE → 还原+异构化), 论文通路图里应在 LOX 通路相邻。AA-CYP/sEH 家族只保留真正的 CYP/sEH 产物 (EpETrE + DiHETrE + 16/18-HETE ω-羟化)。修正后 80 主轨候选 12-HETrE 正确归 AA-LOX, 与"AA-LOX 家族 GSEA q<0.05 显著上调"叙述一致。
- **酶活性比值通路核查导致 v1 → v2 重写（2026-05-19）**：用户审视后发现旧 6 比值有 4 处通路归属错误。**关键教训**：靶向 LC-MS panel 注释（如 family_map 的 enzyme 列）只是"经典酶系"标签, 不等于"该分子主要由该酶生成"; 必须区分**结构唯一可达**（PG/TX/epoxide/diol = level 5, 自由基不可达）与**酶+非酶混合主导**（11/8-HETE, 9-HODE, 7/10/11/13-HDoHE = level 1, 自由基主导）。新版 11 比值按 COX/LOX/CYP/sEH 四酶严格组织, 比值组件全用 evidence_level≥4 的分子, panel 内非完整位置异构体配对（如 5,6-EpETrE 缺）的比值直接弃用。**附带方法学改进**：从 ComBat 校正矩阵反 log2 求和改为 raw 浓度求和；ANCOVA 加 batch 哑变量替代依赖 ComBat 校正——比值层面分子分母的乘性 batch 偏移自然相消, 这才是 ratio 的正确代数形式。
- **数据来源混合的处理范式（2026-05-19）**：主分析矩阵基于供应商 114 panel, 但发现 ori_allmet_QC.xlsx (673 met 全表) 含 EPA/DHA/17-HDoHE/4-HDoHE 等被剔分子。**不重做 ComBat**, 而是 raw 浓度直接抽取需要的分子加入比值计算 (15b/15c)。理由: ComBat 重做会改变所有下游单分子结果与家族 GSEA 的可重现性, 工作量与不确定性巨大; 比值层面 raw 浓度计算 + ANCOVA 加 batch 协变量在化学计量上更干净。这建立了"主分析锁定 / 补充分析按需扩展"的处理范式。
- **16-HDoHE 重归非酶决策（2026-05-19, 见 §7B.5b 完整记录）**：双重证据驱动 — (1) 文献 VanRollins 2008 + Yin & Porter 2005 一致指向 CYP4 ω-7 不偏好 + DHA 自由基 C15→C16 + 17E 重排契合 panel 命名; (2) 实证 9 HDoHE 单分子 ANCOVA 显示 16-HDoHE log2FC+0.173 与非酶组 (7/10/11/13-HDoHE 中位 +0.17) 同质, 远低于纯 CYP 20-HDoHE (+0.333); 剔除后 CYP 比值 FC 反而 1.31×→1.35× 增强 (信号去除稀释)。**附带的方法学价值**：当 panel/数据库注释和文献/数据冲突时, **文献机制 + 数据聚类双重一致**才是可靠的重归属依据, 单一来源不够。该决策同步更新 family_map (16-HDoHE+7/10/11/13-HDoHE 全部从 LOX 改 Non-enzymatic) 和 §7B.5 CYP 比值 (改 20-HDoHE/DHA 单分子), 已落盘到 12 + 15 + 15b 三个脚本和 metabolite_family_map.csv。

- **PUFA 单分子不显著 + 家族 GSEA 显著的解读范式（2026-05-19, §7B.5c）**：7 游离 PUFA 单分子 log2FC 集中在 −0.08 ~ −0.13 极窄区间, 方向 100% 一致 ↓ 6-13%, 但 7 个单分子全部 p>0.05; 而家族 GSEA q=0.004 ↓↓ (7 大家族最强信号). **这不是矛盾, 而是 GSEA 设计上就要捕获的 "small consistent effects accumulate" 模式**: 单分子层因效应量小+样本量限制达不到 p<0.05, 但多分子方向一致提供独立验证的累积证据. **论文写作策略**: 报告 GSEA q=0.004 作为家族级主结论, 同时用 PUFA 单分子表显示"7/7 方向一致 + log2FC 极度同质"作为可读性的方向性证据, 避免审稿人质疑"为什么单分子没显著但家族富集了". DHA 浓度 ↓6% 与其下游氧化产物 ↑ 25-35% 在质量平衡上完整闭环, 强化"底物→产物代谢通量加速"的机制叙述. **教训**: 单分子 BH-q 全不显著 ≠ 没有生物学信号; 多通路 / 多组学 / 多家族联合证据链是观察性研究的标准做法.
- **PUFA 双键数与非酶氧化敏感性的梯度效应：内部对照排除技术伪影（2026-05-20, §7B.5e）**：三大底物池非酶氧化通路 ANCOVA 显示完美的 PUFA 双键数梯度——DHA (22:6, 5 双烯丙位) log2FC=+0.31 p=0.012 ✅; AA (20:4, 3 双烯丙位) +0.22 p=0.032 ✅; **LA (18:2, 1 双烯丙位) +0.06 p=0.523 ❌**. 与 Yin & Porter 2005 自由基氧化速率指数律 (rate ∝ exp(n_bisallylic_H)) 完美吻合. **方法学意义**: 该梯度提供了**内部对照**——如果 AA/DHA 非酶氧化 ↑ 是技术伪影或基线漂移, LA 应同方向变化, 实际 LA 完全不动. 这反向证伪了"全 PUFA 池无差别氧化"的伪影假说, 证实是**孕前肥胖介导胎盘 ROS 选择性攻击高度不饱和 PUFA**的真实生物学. **论文写作策略**: 在 Discussion 中明示三池非酶氧化梯度作为"机制特异性 + 技术稳健性"的双重证据, 配 Yin & Porter 2005 / Niki 2009 文献支撑. **教训**: 在观察性研究中, 设计或事后识别**自然的"阳性 + 阴性 + 阴性"分组对照**, 比单纯报告"显著"更有说服力——因为反例的不显著本身是证据.
- **底物特异性酶系偏好重塑的发现（2026-05-20, §7B.5d/e）**：三大底物池横向对比揭示孕前肥胖背景下**胎盘氧化脂质酶系底物偏好显著重塑**：CYP4-ω 在 AA 池反向 ↓ (p=0.71) 而 DHA 池 ↑↑ (p=0.008) → **CYP4F2/CYP4A11 优先消耗 DHA** (Westphal 2015 PNAS 一致); LOX 在 AA 池接近显著 ↑ (12/15 p≈0.09) 而 LA 池不显著 (p=0.42) → **ALOX12/15 底物偏好 AA >> LA**; CYP-Epo 在 LA 池显著 ↑ (p=0.040) 而 AA 池不显著 → **CYP2C/2J 对 LA 偏好高于 AA**. 这种"每个酶都有偏好底物 + 偏好底物在肥胖背景下被优先消耗"的图景, 是单一通路分析无法捕获的**跨底物机制发现**, 也是论文 Discussion 的核心理论价值. **写作策略**: 用"底物 × 酶系"二维表呈现 (见 §7B.5e), 配各酶亚家族底物 Km 文献支撑.
