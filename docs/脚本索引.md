## 4. 已实现脚本说明

### 4.1 `00_baseline_table1.py`

用途：生成主分析队列 n=165 的临床基线表。

主要功能：

- 连续变量按组做 Shapiro-Wilk 正态性判断。
- 正态且方差齐时用 t 检验，方差不齐时用 Welch t 检验，非正态时用 Mann-Whitney U。
- 分类变量按期望频数选择卡方检验或 Fisher 精确检验。
- 计算连续变量和多分类变量的 SMD，并用阈值标记不均衡程度。
- 输出 Excel 和 Markdown 两种格式。

注意事项：

- 当前脚本输入路径是裸文件名，运行位置会影响能否找到输入文件。
- 当前脚本输出也是相对当前工作目录写出；建议后续统一改为 `ROOT / "results" / "tables"`。

### 4.2 `00b_align_omics_to_cohort.py`

用途：把原始组学矩阵从 198 个样本筛选并重排到临床主分析队列 n=165。

关键规则：

- 临床标本号到组学样本 ID 的映射规则为 `19TPT0XXX -> A19-XXX`、`20TPT0XXX -> B20-XXX`、`21TPT0XXX -> C21-XXX`。
- 保留代谢物元信息列、165 个主队列样本列和浓度单位列。
- 额外输出 `sample_alignment_n165.csv`，用于审计样本 ID、组学 ID 和 BMI 分组。

下游依赖：

- `01_feature_filtering.py` 直接读取 `ori_n165_aligned.xlsx` 和 `sample_alignment_n165.csv`。

### 4.3 `01_feature_filtering.py`

用途：根据分组检出率过滤代谢物，建立主分析和探索性分析双轨。

过滤策略：

- 80% 主分析轨：至少一个 BMI 分组检出率达到 80%，保留为主分析特征。
- 50% 探索轨：至少一个 BMI 分组检出率达到 50%，用于探索性保留边缘低丰度特征。
- 缺失定义为 `NaN` 或 `0`。

输出结果：

- `ori_n165_filtered80.xlsx`：主分析输入。
- `ori_n165_filtered50.xlsx`：探索性输入。
- `feature_filtering_audit.csv`：每个代谢物的检出率、缺失率和去留决定。

### 4.4 `02_imputation.py`

用途：对过滤后的主分析轨和探索轨分别进行缺失机制诊断和 QRILC 插补。

核心流程：

- 计算每个特征的缺失率与观测中位数的 Spearman 相关。
- 若低丰度特征更容易缺失，则判定为 MNAR / LOD 截断信号。
- 在 log2 尺度上按样本执行 QRILC 左截断正态插补，再还原到原始尺度。
- 输出插补审计表，记录插补数量、MNAR 诊断结果和插补值范围。

### 4.5 `03_log_pareto.py`

用途：生成统计模型和无监督可视化所需的转换矩阵。

输出分工：

- `*_log2.xlsx`：用于 ANCOVA / limma / GLM 等统计模型，系数可解释为 log2 fold change。
- `*_log2_pareto.xlsx`：用于 PCA、热图、聚类等无监督分析。

审计内容：

- log2 前后 Shapiro-Wilk 正态性通过率。
- log2 值范围。
- Pareto 缩放后每个特征的行均值和行标准差范围。

### 4.6 `04_batch_effect_diagnosis.py`

用途：诊断采样年份相关的批次效应，并评估年份与 BMI 分组的共线性。

主要分析：

- 对 80% 主分析轨和 50% 探索轨分别运行 PCA。
- 输出按年份和 BMI 分组着色的 PC1-PC2 散点图。
- 对前 5 个 PC 分别计算年份和 BMI 的 ANOVA R² 与 P 值。
- 使用 PERMANOVA 检验年份效应和 BMI 分组效应。
- 计算年份与 BMI 分组的 Cramér's V。

输出位置：

- 图片：`results/figures/01_QC_and_Batch_PCA/`
- 表格：`results/tables/batch_diagnosis_audit.csv`
